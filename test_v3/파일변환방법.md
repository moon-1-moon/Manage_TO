# 통합 정원 파일 변환 및 생성 절차

이 문서는 `(부서명 입력)` 형태의 원본 엑셀 파일들을 최종 `기준정원.xlsx` 및 `운영정원.xlsx` 파일로 변환하는 3단계 절차를 설명합니다.

## 1. 사전 준비 (데이터 전처리)
**목적**: 원본 엑셀 파일(`...부서명 입력...`)에서 필요한 데이터를 추출하여 중간 데이터 파일(`..._데이터.xlsx`)로 변환합니다.

### 실행 위치
`d:\Workspace\Manage_TO\test_v3`

### 실행 스크립트
다음 6개의 파이썬 스크립트를 순차적으로 실행합니다:
1. `python transform_본부_기준정원.py`
2. `python transform_본부_운영정원.py`
3. `python transform_본부_한시정원.py`
4. `python transform_소속기관_기준정원.py`
5. `python transform_소속기관_운영정원.py`
6. `python transform_소속기관_한시정원.py`

### 결과물
- `(251001)본부_기준정원_데이터.xlsx` 등 6개 파일 생성

---

## 2. 한시정원 통합 (합본 생성)
**목적**: 기준/운영 정원 데이터에 한시정원 데이터를 병합하여, 기관별(본부/소속기관) 합본 파일을 생성합니다.

### 실행 위치
`d:\Workspace\Manage_TO\test_v3`

### 실행 스크립트
다음 3개의 파이썬 스크립트를 실행합니다:
1. `python merge_hq_std_data.py` (본부 기준 + 한시 -> 본부 기준 합본)
2. `python merge_hq_data.py` (본부 운영 + 한시 -> 본부 운영 합본)
3. `python merge_sov_data.py` (소속기관 기준/운영 + 한시 -> 소속기관 기준/운영 합본)

### 결과물
- `(251001)본부_기준정원_합본.xlsx`
- `(251001)본부_운영정원_합본.xlsx`
- `(251001)소속기관_기준정원_합본.xlsx`
- `(251001)소속기관_운영정원_합본.xlsx`

---

## 3. 최종 병합 (기준/운영 정원 생성)
**목적**: 본부 데이터와 소속기관 데이터를 하나로 합쳐 최종 결과물을 생성합니다.

### 실행 위치
`d:\Workspace\Manage_TO\test_v3`

### 실행 스크립트
1. `python merge_excel_files.py`

### 최종 결과물
- **`(251001)기준정원.xlsx`**
- **`(251001)운영정원.xlsx`**
